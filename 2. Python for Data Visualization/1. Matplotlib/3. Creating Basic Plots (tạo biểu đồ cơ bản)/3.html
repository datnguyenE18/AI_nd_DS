Basic Example¶

Let's walk through a very simple example using two numpy arrays. You can also use lists, but most likely you'll be passing numpy arrays or pandas columns (which essentially also behave like arrays).

In [6]:
plt.plot( [1, 2, 3, 6, 10], [1, 4, 7, 8, 8])
Out[6]:
[<matplotlib.lines.Line2D at 0x2699243cf40>]

Một vài tham số về định dạng:

  • tham số màu : r - red, g -green, b - blue, y - yellow, …

  • định dạng đường : - (đường thẳng), -- (đường nét đứt), nếu không truyền tham số là không vẽ đường.

  • định dạng đỉnh : * (đỉnh hình bông tuyết), D (đỉnh hình thoi), s (đỉnh hình vuông), go (chấm tròn), ...

Creating Multiplots on Same Canvas¶

(Vẽ hai biểu đồ cạnh nhau)

plt.subplot(nrows, ncols, plot_number)

Đầu tiên ta sử dụng phương thức subplots() để tạo ra 2 biểu đồ. Sử dụng subplots() các đối tượng Axes sẽ được trả về.

In [17]:
fig, (ax1, ax2) = plt.subplots(1, 2, figsize = (10, 6), sharey = True, dpi = 120)

ax1.plot([1, 2, 3, 4, 5], [1, 2, 3, 4, 10], '-go') # Greendots
ax2.plot([1, 2, 3, 4, 5], [1, 2, 3, 4, 10], '-b*') # Bluestart

ax1.set(title = 'Greendots', xlabel = 'X', ylabel = 'Y', xlim = (0, 6), ylim = (0, 12))
ax2.set(title = 'Bluestart', xlabel = 'X', ylabel = 'Y', xlim = (0, 6), ylim = (0, 12))

plt.show()

Tham số (1,2) trong hàm subplots có thể hiểu là "một hàng, hai cột" - tương ứng bạn sẽ vẽ ra 2 biểu đồ cạnh nhau. Nếu chẳng hạn bạn thay (1,2) bằng (2,2), bạn sẽ được 4 biểu đồ.

Tham số dpi là độ phân giải của hình vẽ.

Tham số sharey = True cho phép chia sẻ thuộc tính giữa 2 trục x và y.

Creating Multi-Objects on Same Plot¶

(Vẽ nhiều đối tượng cùng một biểu đồ)

In [20]:
plt.plot([1, 2, 3, 4, 5], [6, 10, 2, 1, 10], 'go-', label = 'GreenDots')
plt.plot([3, 8, 6], [11, 9, 20], 'b*-', label = 'Bluestars')
plt.plot([2, 6, 5, 7, 9], [11, 2, 6, 9, 1], 'y*-', label = 'GreenDots')
plt.title('Example')
plt.xlabel('X')
plt.ylabel('Y')
plt.legend(loc = 'best') # thêm chú thích
# plt.show()
Out[20]:
<matplotlib.legend.Legend at 0x269927bb220>

  • tham số label trong hàm plt.plot() là để đặt tên cho một điểm, tên này sẽ xuất hiện trong chú thích.

  • plt.xlabel hay plt.ylabel là để đặt tên cho trục x và trục y.

  • plt.title là để đặt tên cho biểu đồ

  • plt.legend sẽ đi tìm các thành phần chứa tham số label và đưa vào chú thích
