# Covert Radia sang :
def rad_to_degrees(x, pos):
    'coverts radians to degrees'
    return round(x * 57.2985, 2)
formatter = FuncFormatter(rad_to_degrees)
plt.gca().xaxis.set_major_formatter(formatter)

plt.grid(linestyle = '--', linewidth = 0.5, alpha = 0.15)
plt.title('Sine and Cisine Waves', fontsize = 14)

Note: Ta s dng plt.figure() t li kch thc, phn gii ca biu v s dng plt.plot() v ln biu .

Phng thc xticks() v yticks() l tng t nhau.

  • Cc tham s : Phng thc ny c 2 tham s chnh l ticks v label

      - ticks : tham s ny truyn vo mt list cc gi tr, cc gi tr ny chnh l cc im c ghi s trn trc x - hay gi tt l number label. Trong v d trn, vi  1 radian = 57.2985 . nh vy ta s c list cc gi tr tng ng bng [0.0, 1.57, 3.14, 4.71, 6.28] , hay n cng bng vi [0, pi/2, pi, 3*pi/2, 2*pi]

  - label : tham s ny l list cc nhn m bn mun t thay cho gi tr ca ticks  trn, nu khng c nhn mc nh l cc gi tr ca ticks

  • Ngoi ra, cc tham s khc c th thm vo l cc tham s ca i tng Text m bn c th thm vo nh:

    • fontsize : tham s ny l kch thc font ca number label

    • rotation : gc lch ca text so vi trc thng ng (trc y)

  • Chuyn radian sang : S dng FuncFormatter thm vo mt hm t nh ngha chuyn radian sang .

Phng thc plt.gca() tr v i tng Axes hin ti, tc l biu hin ti.
