Pandas cung cp mt phng thc gip ta thay i shape ca d liu nh trn l .pivot( index = None, columns = None, values = None ).

th phng thc .pivot() khng th rearrange cc ct ca mt dataframe. Nh vy bi ton hin ti l cn phi chuyn mt vi index level v column level. lm c iu ny th c th s dng phng thc:

.unstack( level = -1, fill_value = None )
