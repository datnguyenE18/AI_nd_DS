Bi tp 2: V th hm tanh

Cng thc hm tanh: $tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$

o hm: $dx = 1 - {tanh(x)}^2$

def tanh(x):
    ex = np.exp(x)
    emx = np.exp(-x)
    return (ex - emx) / (ex + emx)

# Calculate plot points
x = np.arange(-5., 5., 0.1)
a = tanh(x)
dx = 1 - a**2

Bi tp 3: Cng vi nhng hng dn nh trong bi tp 2, cc bn hy t xy dng t u v hm: ReLU

Cng thc hm ReLU:

$f(x) = 0$ nu $x < 0$

$f(x) = x$ nu $x >= 0$

o hm ReLU:

$f'(x) = 0$ nu $x < 0$

$f'(x) = 1$ nu $x >= 0$

In [6]:
### START CODE HERE ### 
#define relu
def relu(x):
  return np.where(x < 0.,0,x)

#define derivative of relu
def derivative_of_relu(x):
  return np.where(x < 0.,0,1)

# Calculate plot points
x = np.arange(-2., 2., 0.005)
a = relu(x)
dx = derivative_of_relu(x)

#create new fig and axes
fig, ax = plt.subplots(figsize=[10,5])

#Setup  axes
ax.spines['left'].set_position(('data',0))
ax.spines['bottom'].set_position(('data',-0.1))
ax.spines['right'].set_color('none')
ax.spines['top'].set_color('none')
ax.xaxis.set_ticks_position('bottom')
ax.yaxis.set_ticks_position('left')

#draw relu
ax.plot(x,a,color = '#d35400',linewidth = 3,label = 'relu')

#draw derivative of relu
ax.plot(x,dx,color = '#1abd15',linewidth = 3,label = 'derivative')

# set legend
ax.legend()
plt.show()
